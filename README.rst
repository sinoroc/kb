Style guide
-----------

Use the following for section headings:

* ``#`` with overline, for parts
* ``*`` with overline, for chapters
* ``=``, for sections
* ``-``, for subsections
* ``^``, for subsubsections
* ``"``, for paragraphs

Suggestion taken from
`Python Developer's Guide <https://devguide.python.org/documentation/markup/#sections>`_.
